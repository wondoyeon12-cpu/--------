# 📝 워드프레스 수동 업로드 1분 컷 가이드

대표님, API 자동화가 워드프레스 보안 설정(또는 호스팅 서버 방화벽)에 막혀서 401 오류를 내뿜고 있습니다.
하지만 걱정 마십시오! 제가 완벽한 HTML 코드를 에디터에 열어드렸으니, **단 1분이면 3개의 글 업로드가 끝납니다.**

### 🎯 수동 업로드 순서 (초간단)

1. 워드프레스 관리자 페이지 접속 👉 **[글] -> [새로 추가]** 클릭
2. 에디터 우측 상단의 **점 3개(옵션) 버튼** 클릭 👉 **[코드 편집기(Code editor)]** 클릭
   *(이게 가장 중요합니다! 비주얼 에디터에 붙여넣으면 코드가 다 깨집니다.)*
3. 제가 바탕화면에 만들어둔 `article_...` .html 파일 열기
4. **파일 안의 소스코드(Ctrl+A로 전체 선택, Ctrl+C로 복사)를 통째로 복사해서, 워드프레스 코드 편집기 창에 붙여넣기(Ctrl+V)**
5. 제목 입력 후 **[임시저장]** 또는 **[공개]** 버튼 클릭!
